Font Family

Adding a font

If you'd like to add a new font from Google Fonts, just add or replace the imported file from scss/theme.scss.

@import url("https://fonts.googleapis.com/css2?family=Open+Sans:ital,wght@0,300;0,400;0,600;0,700;0,800;1,300;1,400;1,600;1,700;1,800&display=swap");

Base font

Update the base font by changing the following variable in scss/_variables.scss:

$font-family-sans-serif: 'Open Sans', "sans-serif;" !default;
